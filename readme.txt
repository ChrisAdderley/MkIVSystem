=============================
MkIV Spaceplane Parts v2.0.0
=============================

This is a parts pack for Kerbal Space Program v1.04, providing a new stockalike spaceplane fuselage system that is designed for hauling very large parts.

============
INSTALLATION
============

To install, place the GameData folder inside your Kerbal Space Program folder. If asked to overwrite files, please do so. REMOVE ANY OLD VERSIONS OF THE MARK IV SYSTEM BEFORE INSTALLING.
